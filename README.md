# Windows Scripts

A collection of scripts for the Command shell and PowerShell.

## PowerShell

### Utilities

| Script | Description |
| --- | --- |
| [Get-FolderSize.ps1](scripts/powershell/Get-FolderSize.ps1) | Lists folders sorted by total disk usage, largest first |
| [Split-File.ps1](scripts/powershell/Split-File.ps1) | Splits or joins files |

### Testing

| Script | Description |
| --- | --- |
| [New-DummyFile.ps1](scripts/powershell/New-DummyFile.ps1) | Creates zero-filled dummy files of a specified size for disk space testing |
| [Start-CpuLoad.ps1](scripts/powershell/Start-CpuLoad.ps1) | Maintains a target CPU usage percentage for a specified duration, for load testing CPU monitors |
| [Start-MemoryLoad.ps1](scripts/powershell/Start-MemoryLoad.ps1) | Maintains a target physical memory usage percentage for a specified duration, for load testing memory monitors |
